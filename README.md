# Nutri JCV

Sistema web local para una clínica de nutrición.

## Funcionalidades

- Login con `sessionStorage`.
- Bloqueo de acceso directo al panel.
- Registro único de pacientes.
- Cálculo automático de IMC.
- Diagnóstico nutricional automático.
- Registro de consultas sin duplicar datos antropométricos.
- Historial clínico cronológico.
- Edición y eliminación de consultas en tiempo real.
- Datos guardados en `localStorage`.

## Cómo correrlo

Usa Visual Studio Code con la extensión Live Server.

1. Abre la carpeta del proyecto.
2. Clic derecho en `index.html`.
3. Selecciona `Open with Live Server`.

No abras el archivo con doble clic, porque los módulos JavaScript pueden fallar.
